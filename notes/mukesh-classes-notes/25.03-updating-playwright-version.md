# Updating Playwright Version — Real Scenario

## Scenario

You're working on an automation framework at your company. One day your lead mentions that Playwright has released a new version with improved auto-waiting, better trace viewer, and a fix for a flaky locator issue your team has been facing. You need to upgrade the framework from v1.48.0 to the latest version.

Here's exactly how you'd do it step by step.

---

## Step 1 — Check Your Current Version

Before doing anything, confirm what version you're currently on.

```bash
npx playwright --version
```

Output might look like: `Version 1.48.0`

---

## Step 2 — Update the Playwright Package

You have two approaches here.

### Approach A — Via Command Line *(recommended)*

To update to the absolute latest version:

```bash
npm install -D @playwright/test@latest
```

Or if your team has decided on a specific version (say your lead said "upgrade to 1.55.0, that's what QA has approved"):

```bash
npm install -D @playwright/test@1.55.0
```

Either command automatically updates `package.json` under `devDependencies` for you.

### Approach B — Manually Edit `package.json`

Open `package.json`, find the `devDependencies` section, and change the version manually:

```json
"devDependencies": {
  "@playwright/test": "1.55.0"
}
```

Then run:

```bash
npm install
```

This will pull the specified version into `node_modules`.

> Both approaches achieve the same result. The command-line approach is faster and avoids typos.

---

## Step 3 — Update Browser Binaries *(critical, often missed)*

This is the step many people forget. Playwright bundles its own browser binaries (Chromium, Firefox, WebKit). When you update the Playwright package, the old browser binaries become incompatible. If you skip this step, your tests will fail with **version mismatch errors**.

To install all browsers:

```bash
npx playwright install
```

Or if your framework only runs on Chromium and you want to save time and disk space:

```bash
npx playwright install chromium
```

---

## Step 4 — Verify the Update

```bash
npx playwright --version
```

Output should now show: `Version 1.55.0` (or whatever version you installed).

At this point, run your test suite once to make sure nothing is broken due to any deprecations or API changes in the new version.
