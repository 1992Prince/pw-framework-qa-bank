# Excel File Reading in Playwright — Complete Notes

## 1) JavaScript ko Excel Files ke Liye Native Support Nahi Hai

JavaScript (aur Node.js) natively sirf text-based files ko read kar sakta hai — jaise `.txt`, `.json`, `.csv`. Ye sab plain text files hote hain jo human readable hote hain.

Excel files (`.xlsx`) binary format mein hote hain — jo padne ke liye special parsing library chahiye. JavaScript ke paas khud se Excel ko samjhne ki taaqat nahi hai. Isliye hume third-party library use karna padta hai — jaise **ExcelJS**.

```
Text file (.txt)     ← Node.js directly padh sakta hai
JSON file (.json)    ← Node.js directly padh sakta hai  
CSV file (.csv)      ← Node.js padh sakta hai, lekin parse karne ke liye library chahiye (PapaParse)
Excel file (.xlsx)   ← Node.js seedha nahi padh sakta, ExcelJS library mandatory hai
```

---

## 2) Excel Se Data Padhne Ke Liye Do Approaches Hain

### Approach 1 — Excel → CSV → PapaParse:
Excel file ko ek baar manually Excel mein khol kar CSV format mein save karo. Phir CSV ko PapaParse se parse karo.

- **Fayde:** Ek hi library (PapaParse) use hona padega, Excel library ki zaroorat nahi.
- **Nuksaan:** Extra manual step — har baar data update hone par Excel se CSV mein convert karna padega.

### Approach 2 — Direct Excel → ExcelJS:
ExcelJS library use karo jo directly Excel files ko read kar sakta hai. Koi conversion ki zaroorat nahi.

- **Fayde:** Automatic hai — Excel file mein koi bhi change hoga immediately reflect hoga.
- **Nuksaan:** Ek aur library add karna padega.

> Real projects mein **Approach 2** use hote hain kyunki yeh automatic aur maintainable hai.

---

## 3) Excel File Padhne Ke Liye 2 Zaroori Cheezein

1. **File Path** — Excel file ki location. Example: `"testdata/registration.xlsx"` ya `"C:/data/users.xlsx"`
2. **Sheet Name** — kaun si sheet se data padhna hai. Example: `'Sheet1'`, `'Registration'`, `'TestData'`

Ek Excel file mein multiple sheets ho sakti hain (jaise workbook mein multiple sheets). Hume specify karna padta hai kaun si sheet se data chahiye.

```javascript
const sheet = workbook.getWorksheet('Sheet1');  // 'Sheet1' naam ki sheet uthao
```

## 4) ExcelJS Library Install Karo

```bash
npm install -D exceljs
```

Yeh command:
- npm registry se exceljs library download karega
- Isko `node_modules/` folder mein save karega
- `package.json` ke `devDependencies` mein add karega:

```json
"devDependencies": {
  "exceljs": "^4.4.0"
}
```

> `-D` flag ka matlab — yeh library sirf development/testing ke liye chahiye, production mein nahi.

## 5) package.json Mein `"type": "module"` Zaroor Likho

`package.json` mein yeh line likha hona chahiye:

```json
{
  "name": "pw-fusion",
  "type": "module",
  "version": "1.0.0",
  "devDependencies": { ... }
}
```

### Kyun zaroori hai?

JavaScript ke do module systems hain — **CommonJS** aur **ES6 Modules**.

**CommonJS (purana tarika):**
```javascript
const ExcelJS = require("exceljs");  // require() syntax use karte hain
```

**ES6 Modules (naya tarika):**
```javascript
import ExcelJS from "exceljs";  // import syntax use karte hain
```

Playwright aur modern frameworks ES6 Modules use karte hain (`import` syntax). Agar `package.json` mein `"type": "module"` nahi hai to Node.js samjhta hai ki humne CommonJS use karna hai (`require()` syntax). Aur jab `import` likha hota hai CommonJS project mein, toh error aata hai:

```
SyntaxError: Cannot use import statement outside a module
```

> **Translation:** "import statement sirf module format mein use ho sakta hai, lekin tumhara project CommonJS format mein hai."

---

## 6) import vs require() — Kyun Alag Hain Aur Best Practice Kya Hai?

### CommonJS — `require()` (Purana)

```javascript
const ExcelJS = require("exceljs");
const fs = require("fs");
```

**Kaisa kaam karta hai:**
- **Synchronous** — file tou tou tak wait karta hai jab tak code load nahi ho jata
- **Runtime pe load hota hai** — jab `require()` line execute hoti hai tab library load hoti hai
- Poori library poore time memory mein rahti hai

**Nuksaan:**
- Slow — especially large files ke liye
- Large projects mein code ke starting se sara load ho jata hai

---

### ES6 Modules — `import` (Naya)

```javascript
import ExcelJS from "exceljs";
import fs from "fs";
```

**Kaisa kaam karta hai:**
- **Asynchronous** — file load karte wakt script execute karta rahta hai (non-blocking)
- **Static imports** — file ke top pe likha hota hai, pehle se parse hota hai
- **Tree-shaking possible** — unused code automatically remove hota hai bundling mein

**Fayde:**
- Fast — parallel loading
- Clean syntax
- Modern JavaScript standard

---

### Best Practice — Kaunsa Use Kare?

**Sirf `import` use karo.** Kyun?

- Playwright aur modern frameworks ES6 Modules expect karte hain
- Better performance — asynchronous loading
- Industry standard — sab modern projects `import` use karte hain
- Future-proof — CommonJS ab deprecated ho raha hai

> Matlab `require()` kabhi use mat karo. Sirf `import` likho.

---

### Summary Table — import vs require()

| Feature | `require()` (CommonJS) | `import` (ES6) |
|---|---|---|
| Syntax | `const X = require("lib")` | `import X from "lib"` |
| Kab load hota hai | Runtime (code execute hote waqt) | Build time (file parse hote waqt) |
| Synchronous ya Async | Synchronous | Asynchronous |
| Performance | Slow (sirf ek ek karke load) | Fast (parallel load) |
| Package.json setting | `"type": "commonjs"` (ya likha nahi hai) | `"type": "module"` |
| Best practice | ❌ Use mat karo | ✅ Hamesha use karo |

---

### Practical Example

**Wrong — CommonJS style (mat likho):**
```javascript
const ExcelJS = require("exceljs");  // ❌ purana tarika
const fs = require("fs");             // ❌ purana tarika
```

**Right — ES6 Modules style (likho):**
```javascript
import ExcelJS from "exceljs";  // ✅ naya tarika
import fs from "fs";             // ✅ naya tarika
```

## Excel File Reading — Complete Implementation and Approach

### Step 1 — ExcelJS Module Import Karo

```javascript
import ExcelJS from "exceljs";
```

**Kya hota hai?**
- `ExcelJS` ek library/class hai jo Excel files handle karta hai
- Haan, naam change kar sakte ho — `import ExcelJS as XLS from "exceljs"` likho toh `XLS` ban jayega
- Lekin convention mein `ExcelJS` hi likha jata hai kyunki yeh library ka actual name hai

---

### Step 2 — Workbook Instance Banao

```javascript
const workbook = new ExcelJS.Workbook();
```

**Kya hota hai?**
- `new ExcelJS.Workbook()` — ExcelJS class se ek naya workbook object banaya
- `workbook` ek empty container hai jisme soon Excel file ka data load hoga
- Filhaal empty hai, abhi file read nahi kiye

---

### Step 3 — Excel File Ko Workbook Mein Load Karo

```javascript
await workbook.xlsx.readFile("testdata/registration.xlsx");
```

**Kya hota hai?**
- `readFile()` — disk se file padh ke workbook mein load kar deta hai
- `await` — kyunki yeh async operation hai, file padne mein time lagta hai
- Ab `workbook` mein poori Excel file ka data hai (sab sheets, sab rows)

---

### Step 4 — Specific Sheet Nikalo

```javascript
const sheet = workbook.getWorksheet('Sheet1');
```

**Kya hota hai?**
- `getWorksheet('Sheet1')` — workbook mein se `'Sheet1'` naam ki sheet nikali
- Agar tumhara sheet `'Registration'` ya `'TestData'` naam ka hai toh wo likho
- `sheet` mein ab sirf Sheet1 ka data hai

---

### Step 5 — Headers Fetch Karo

```javascript
const headerRow = sheet.getRow(1);         
const headers = headerRow.values.slice(1); 
console.log(headers);
```

**Line-by-line kya hota hai:**

```javascript
const headerRow = sheet.getRow(1);
```
- `getRow(1)` — sheet ke first row ko uthaya (row number 1)
- `headerRow` ek object hai jo puri row ka data rakhta hai
- `headerRow` kya hota hai? Ek object with properties like `values`, `number`, etc.

```javascript
const headers = headerRow.values.slice(1);
```
- `headerRow.values` — ek array hai jo sare cell values ko represent karta hai
- ExcelJS ka quirk: index 0 always empty hota hai (ExcelJS use 1-based indexing)
- `slice(1)` — index 0 ko hatane ke liye, sirf actual headers lene ke liye
- Result: Ek simple array of strings

**Console output:**
```javascript
[ 'id', 'name', 'email', 'password' ]
```

> So `headers` ek array of strings hai — keys hain jo object mein use honge.

---

### Step 6 — Empty Array Declare Karo Data Store Karne Ke Liye

```javascript
const testdata = [];
```

Yeh array mein sare Excel rows ko objects mein convert karke push karenege.

---

- Excel file ko read kiya aur workbook mein load kiya
- Sheet1 nikali
- Headers array banaya jo Excel ke first row se aaya: `['id', 'name', 'email', 'password']`

## Excel Data Reading Strategy

### Step 2 — TestData Array Declare Karo

```javascript
const testdata = [];
```

Yeh ek empty array hai jo ultimately array of objects banegi. Har object ek Excel row ko represent karega.

---

### Step 3 — sheet.eachRow() Method Use Karo

```javascript
sheet.eachRow((row, rowNum) => {
  // code yahan
});
```

**Kya hota hai?**
- `sheet.eachRow()` — ExcelJS ka built-in method jo sare rows ko iterate karta hai
- `row` parameter — current row ka complete data (sare cells)
- `rowNum` parameter — current row ka number (1, 2, 3, ...)

Har iteration mein:
- Iteration 1: `rowNum = 1` (headers)
- Iteration 2: `rowNum = 2` (first data row)
- Iteration 3: `rowNum = 3` (second data row)
- Iteration 4: `rowNum = 4` (third data row)
- ...aur aise

---

### Step 4 — First Row Skip Karo (Headers)

```javascript
if (rowNum === 1) return;
```

**Kyun?**
- `rowNum = 1` matlab headers row hai
- Headers ko skip karna hai kyunki hume sirf data rows chahiye
- `return` statement — loop ka yeh iteration skip kar dega aur next row pe jaayega

---

### Step 5 — Current Row Ke Liye Blank Object Banao

```javascript
const obj = {};
```

**Kya hota hai?**
- Har iteration (har row) ke liye ek naya blank object banate hain
- Isme soon Excel ke us row ka data aayega

Example:
- Iteration 2 mein: `obj` mein TC001 wala row data aayega
- Iteration 3 mein: naya `obj` banega jo John wala row data rakhega
- Aur aise...

---

### Step 6 — Headers Array Ko Loop Karo aur Object Ko Fill Karo

```javascript
headers.forEach((header, index) => {
  obj[header] = String(row.values[index + 1] ?? "");
});
```

**Step by step kya hota hai:**

`headers.forEach((header, index) => { })` — Headers array ko loop karta hai:
- Iteration 1: `header = 'id'`, `index = 0`
- Iteration 2: `header = 'name'`, `index = 1`
- Iteration 3: `header = 'email'`, `index = 2`
- Iteration 4: `header = 'password'`, `index = 3`

`obj[header] = ...` — Object mein key-value pair set karta hai:
- `obj['id'] = ...`
- `obj['name'] = ...`
- `obj['email'] = ...`
- `obj['password'] = ...`

`String(row.values[index + 1] ?? "")` — Excel sheet se cell value nikalta hai:

**`row.values` kya hota hai?**
```javascript
// Excel row: [A2: TC001] [B2: Sarika] [C2: sarika@example.com] [D2: password123]
row.values = [undefined, 'TC001', 'Sarika', 'sarika@example.com', 'password123']
             // 0          1          2             3                  4
```

ExcelJS ke `row.values` mein index 0 always `undefined` hota hai. Isliye:
- `index = 0` (header `'id'`) → `row.values[0 + 1]` = `row.values[1]` = `'TC001'`
- `index = 1` (header `'name'`) → `row.values[1 + 1]` = `row.values[2]` = `'Sarika'`
- `index = 2` (header `'email'`) → `row.values[2 + 1]` = `row.values[3]` = `'sarika@example.com'`
- `index = 3` (header `'password'`) → `row.values[3 + 1]` = `row.values[4]` = `'password123'`

- `?? ""` — agar cell value `null` ya `undefined` hai toh empty string `""` use karo.
- `String(...)` — cell value ko string mein convert karo.

---

### Interview Question: "`obj[header] = ...` Se Kaise Object Banata Hai?"

**Answer:**

"`obj[header]` dynamic key assignment karta hai. `header` variable hai jo `'id'`, `'name'`, `'email'`, `'password'` change hota hai."

```javascript
header = 'id'
obj[header] = 'TC001'     // yeh likha jata hai obj['id'] = 'TC001'

header = 'name'
obj[header] = 'Sarika'    // yeh likha jata hai obj['name'] = 'Sarika'

header = 'email'
obj[header] = 'sarika@example.com'  // yeh likha jata hai obj['email'] = 'sarika@example.com'
```

Toh `forEach` loop complete hone ke baad:
```javascript
obj = {
  id: 'TC001',
  name: 'Sarika',
  email: 'sarika@example.com',
  password: 'password123'
}
```

"Har key ek header hai, har value Excel cell se nikla hua hai."

---

### Step 7 — Object Ko TestData Array Mein Push Karo

```javascript
testdata.push(obj);
```

**Kya hota hai?**
- Poora object (`obj`) ko `testdata` array mein add kar diya
- Ab yeh object array mein permanently store ho gaya
- Next iteration mein naya `obj` banega

**Loop ke baad `testdata` kya hoga:**
```javascript
testdata = [
  { id: 'TC001', name: 'Sarika', email: 'sarika@example.com', password: 'password123' },
  { id: 'TC002', name: 'John',   email: 'john@example.com',   password: 'password456' },
  { id: 'TC003', name: 'Alice',  email: 'alice@example.com',  password: 'password789' }
]
```

## Complete Code

```javascript
import ExcelJS from "exceljs";

async function readExcelData() {
  // STEP 1: Workbook, Sheet, Headers Setup
  const workbook = new ExcelJS.Workbook();
  await workbook.xlsx.readFile("testdata/registration.xlsx");

  const sheet = workbook.getWorksheet('Sheet1');

  const headerRow = sheet.getRow(1);
  const headers = headerRow.values.slice(1);  // ['id', 'name', 'email', 'password']

  // STEP 2: TestData Array Declare
  const testdata = [];

  // STEP 3: Sheet Ke Sare Rows Ko Iterate Karo
  sheet.eachRow((row, rowNum) => {
    
    // STEP 4: First Row Skip Karo
    if (rowNum === 1) return;

    // STEP 5: Current Row Ke Liye Blank Object Banao
    const obj = {};

    // STEP 6: Headers Array Ko Loop Karo aur Object Ko Fill Karo
    headers.forEach((header, index) => {
      // obj['id'] = 'TC001'
      // obj['name'] = 'Sarika'
      // obj['email'] = 'sarika@example.com'
      // obj['password'] = 'password123'
      obj[header] = String(row.values[index + 1] ?? "");
    });

    // STEP 7: Object Ko TestData Array Mein Push Karo
    testdata.push(obj);
  });

  // Final Result
  console.log(testdata);
  return testdata;
}

// Function Call Karo
await readExcelData();
```
