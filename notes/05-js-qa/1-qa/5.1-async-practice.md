# Practice 1 — Basic Promise

### Problem

Create a Promise that resolves with your name after 1 second.
Log the name in `.then()`. Log `"Done"` in `.finally()`.

### Solution

```javascript
const myPromise = new Promise((resolve) => {

    setTimeout(() => {
        resolve("Prince");
    }, 1000);

});

myPromise
    .then((name) => {
        console.log(name);
    })
    .finally(() => {
        console.log("Done");
    });
```

# Practice 2 — Resolve and Reject

### Problem

Create a function `checkAge(age)` that returns a Promise.

- If `age >= 18`, resolve with `"Access granted"`.
- If `age < 18`, reject with `"Access denied"`.
- Handle both cases using `.then()` and `.catch()`.

### Solution

```javascript
function checkAge(age) {

    return new Promise((resolve, reject) => {

        if (age >= 18) {
            resolve("Access granted");
        } else {
            reject("Access denied");
        }

    });

}

checkAge(20)
    .then((message) => {
        console.log(message);
    })
    .catch((error) => {
        console.log(error);
    });

checkAge(15)
    .then((message) => {
        console.log(message);
    })
    .catch((error) => {
        console.log(error);
    });
```

# Practice 3 — Object through Promise

### Problem

Create a Promise that resolves with the following object:

```javascript
{
    id: 101,
    product: "Keyboard",
    price: 1500
}
```

In `.then()`, log the product name and price separately.

### Solution

```javascript
const productPromise = new Promise((resolve) => {

    resolve({
        id: 101,
        product: "Keyboard",
        price: 1500
    });

});

productPromise.then((product) => {

    console.log(product.product);
    console.log(product.price);

});
```

# Practice 4 — Promise Chaining

### Problem

Write two functions:

- `fetchProduct()` → resolves with `{ id: 5, name: "Monitor" }` after 1 second.
- `fetchReviews(id)` → resolves with `["Great!", "Value for money"]` after 1 second.

Chain them:

- First fetch the product.
- Then fetch its reviews using `product.id`.
- Log both results.
- Handle errors.
- Log `"Complete"` in `.finally()`.

### Solution

```javascript
function fetchProduct() {

    return new Promise((resolve) => {

        setTimeout(() => {

            resolve({
                id: 5,
                name: "Monitor"
            });

        }, 1000);

    });

}

function fetchReviews(id) {

    return new Promise((resolve) => {

        setTimeout(() => {

            resolve(["Great!", "Value for money"]);

        }, 1000);

    });

}

fetchProduct()
    .then((product) => {

        console.log(product);

        return fetchReviews(product.id);

    })
    .then((reviews) => {

        console.log(reviews);

    })
    .catch((error) => {

        console.log(error);

    })
    .finally(() => {

        console.log("Complete");

    });
```

# Practice 5 — finally() Use Case

### Problem

Simulate a form submission.

- Set `isSubmitting = true` before creating the Promise.
- Resolve the Promise with `"Form saved!"`.
- In `.then()`, log the success message.
- In `.finally()`, set `isSubmitting = false` and log `"Button re-enabled"`.

### Solution

```javascript
let isSubmitting = true;

const submitForm = new Promise((resolve) => {

    setTimeout(() => {

        resolve("Form saved!");

    }, 1000);

});

submitForm
    .then((message) => {

        console.log(message);

    })
    .finally(() => {

        isSubmitting = false;

        console.log("Button re-enabled");
        console.log(isSubmitting);

    });
```

# Practice 6 — Promise.resolve() Shorthand

### Problem

Use `Promise.resolve()` to create an already fulfilled Promise with value `42`.

- In the first `.then()`, log the value doubled.
- In the second `.then()`, log `"Done"`.

### Solution

```javascript
Promise.resolve(42)
    .then((value) => {

        console.log(value * 2);

    })
    .then(() => {

        console.log("Done");

    });
```
